પ્રશ્નોના મૌખિક જવાબો (પ્ર.501) … ( વિક્ષેપો ) શ્રી જે.સી. દિવાકર રેડ્ડી: મેડમ સ્પીકર, મને આ તક આપવા બદલ તમારો ખૂબ ખૂબ આભાર. … ( વિક્ષેપો ) આખું ઓગસ્ટ હાઉસ મારી સાથે સંમત થશે કે અમારા માનનીયના ગતિશીલ નેતૃત્વ હેઠળ. વડા પ્રધાન 'મેક ઇન ઇન્ડિયા'નું સપનું અને આપણા માનનીય ઉદ્દેશ્ય. આંધ્રપ્રદેશના મુખ્યમંત્રી - જેઓ હંમેશા 'મેક ઇન આંધ્ર' ના સ્વપ્નને સાકાર કરવા પ્રયત્નશીલ રહે છે - મને આશા છે કે, નેશનલ ઈન્ડસ્ટ્રીયલ એન્ડ મેન્યુફેક્ચરિંગ ઝોન્સ (NIMZs) નિઃશંકપણે ભવિષ્યમાં સફળ થશે.… ( વિક્ષેપો ) નેશનલ મેન્યુફેક્ચરિંગ પોલિસીમાં ઉત્પાદન ઉદ્યોગને રાષ્ટ્રીય રોકાણ અને ઉત્પાદન ક્ષેત્રોમાં અને સમગ્ર દેશમાં મેન્યુફેક્ચરિંગ ઉદ્યોગને સગવડ આપવા માટે વિતરણનો સમાવેશ થાય છે, જેમાં મેન્યુફેક્ચરિંગ એકમોને પોતાને ક્લસ્ટરમાં ગોઠવવામાં સક્ષમ બનાવવાનો સમાવેશ થાય છે. … ( વિક્ષેપો ) નાણાકીય વર્ષ 2016-17 માટે રૂ. 3.35 કરોડ રાષ્ટ્રીય ઉત્પાદન નીતિના અમલીકરણ, NIMZ ના માસ્ટર પ્લાનિંગ અને ટેક્નોલોજી એક્વિઝિશન એન્ડ ડેવલપમેન્ટ ફંડ હેઠળ રાખવામાં આવ્યા છે. … ( વિક્ષેપો ) માનનીય માટે મારો પ્રથમ પૂરક પ્રશ્ન. મંત્રી, તમારા થકી, મેડમ સ્પીકર આ છે. આ NIMZ ક્યારે દિવસનો પ્રકાશ જોશે? જેમ તમે જાણો છો કે અનંતપુર એ સૂચિત બેંગલુરુ-કુર્નૂલ ઇન્ડસ્ટ્રીયલ કોરિડોર અને CBIC એટલે કે ચેન્નાઇ-બેંગલુરુ ઇન્ડસ્ટ્રીયલ કોરિડોરનો એક ભાગ છે. એનઆઈએમઝેડમાં અનંતપુરનો સમાવેશ કરવાની કોઈ દરખાસ્ત છે? શું આ 22 NIMZ ના સંપૂર્ણ અમલીકરણ માટે કોઈ કાર્ય યોજના અથવા સમયમર્યાદા નક્કી કરવામાં આવી છે? … ( વિક્ષેપો ) શ્રીમતી નિર્મલા સીતારમણ: મેડમ સ્પીકર, માનનીય. સભ્યએ અનંતપુર જિલ્લા વિશે પ્રશ્નો પૂછ્યા છે. … ( વિક્ષેપો ) જેમ કે, અમે જાણીએ છીએ કે દિલ્હી-મુંબઈ ઇન્ડસ્ટ્રીયલ કોરિડોર હેઠળના આઠ ઝોનને આવરી લેવામાં આવ્યા છે અને અન્ય 14, જે દિલ્હી-મુંબઈ ઇન્ડસ્ટ્રીયલ કોરિડોરની બહાર છે, ઝોન તરીકે સમર્પિત છે, પરંતુ ત્યાં કોઈ નવી દરખાસ્ત નથી. ચોક્કસ જિલ્લાનો સમાવેશ કરો, જેમ કે સભ્યએ અનંતપુરને ટેકો આપ્યો છે. તે અથવા અન્યને સૂચિમાં નવા ઉમેરવામાં આવ્યા નથી. … ( વિક્ષેપો ) શ્રી જે.સી. દિવાકર રેડ્ડી: મેડમ, અનંતપુર એક પછાત જિલ્લો છે. તે બેંગલુરુની બાજુમાં છે અને બેંગલુરુની નજીક છે. તેથી, માન. મંત્રીએ અનંતપુરના વિકાસની ચિંતા કરવાની છે. … ( વિક્ષેપો ) હું તેણીને ત્યાં વધુ એક ઝોનની દરખાસ્ત કરવા વિનંતી કરીશ જેથી ત્યાં કાપડ અને કૃષિ જેવા ઉદ્યોગો સ્થાપવાની તક મળે. તેથી, માનનીયને મારી ખાસ વિનંતી છે. મંત્રી અનંતપુરનો સમાવેશ કરવાનો પ્રયાસ કરે. … ( વિક્ષેપો ) ) પૂ. સ્પીકર: મને માફ કરશો. … ( વિક્ષેપો ) પૂ. સ્પીકર: 11.20 વાગ્યે ફરીથી બેઠક માટે ગૃહની કાર્યવાહી સ્થગિત કરવામાં આવી ત્યારબાદ લોકસભા અગિયાર વાગ્યાના વીસ મિનિટ સુધી સ્થગિત કરી દેવામાં આવી હતી ઘડિયાળની. પ્રશ્નોના મૌખિક જવાબો.... ચાલુ. શ્રી જે.સી. દિવાકર રેડ્ડી: મેડમ, આદરણીય મંત્રી અનંતપુરને સારી રીતે જાણે છે. … ( વિક્ષેપો ) HON. સ્પીકર: ના, હું કોઈને મંજૂરી આપતો નથી. … ( વિક્ષેપો ) શ્રી જેસી દિવાકર રેડ્ડી: અનંતપુર એક પછાત વિસ્તાર છે. તે રાજસ્થાનના જેસલમેર જિલ્લા કરતાં પણ ખરાબ છે કારણ કે ઔદ્યોગિક અને કૃષિની દ્રષ્ટિએ આપણે ખૂબ ગરીબ છીએ અને ઘણા લોકો ત્યાંથી સ્થળાંતર કરી રહ્યા છે. HON. વક્તા: તમારો પ્રશ્ન શું છે? શ્રી જે.સી. દિવાકર રેડ્ડી: હા, મેડમ, હું પૂછું છું. તેઓ બેંગ્લોર, ચેન્નાઈ અને અન્ય સ્થળોએ સ્થળાંતર કરી રહ્યા છે. દિવસેને દિવસે ત્યાંથી સ્થળાંતર કરનારા લોકોની સમસ્યા વધી રહી છે, પરંતુ સાથે સાથે આપણે ત્યાં ઘણી પડતર જમીન અને સારી જમીન પણ છે. મારી વિનંતી માત્ર આટલી જ છે, તમારા દ્વારા, અમારા માન. મંત્રી. તેણીએ પ્રકાશમ અને ચિત્તૂર જિલ્લાઓ અને ત્યાં આપેલા બે કેન્દ્રોને માન્યતા આપી છે. મારી વિનંતી છે કે મારો જિલ્લો આ બધા જિલ્લાઓ કરતાં ખરાબ છે. તમે અમારા માટે વધુ એક કેન્દ્ર કેમ નથી આપતા? સરકાર કંઈપણ ગુમાવશે નહીં જ્યારે તે આપણને આર્થિક અને આર્થિક રીતે વિકાસ કરવાનો અવકાશ આપશે. કૃપા કરીને તેના માટે મારા જિલ્લાને ધ્યાનમાં લો. શ્રીમતી નિર્મલા સીતારામન: મેડમ સ્પીકર, જે મુદ્દો માન. સદસ્યએ જે ઉઠાવ્યું તે ચોક્કસપણે ખૂબ જ મહત્વપૂર્ણ છે કારણ કે અનંતપુર જિલ્લો ઝડપથી રણ બની રહ્યો છે. અમે ઓળખીએ છીએ કે આ જિલ્લાને મુશ્કેલી છે કારણ કે તેને ઔદ્યોગિક રીતે અને અન્યથા તેના લોકો માટે તકો પૂરી પાડવાની છે. પરંતુ જો રાજ્ય સરકાર અમને કોઈ પ્રકારનો પ્રસ્તાવ મોકલે છે, તો અમે તેને કયા વિભાગમાં જવાનું છે તેના આધારે વિચારી શકીએ છીએ. તેથી, તે રાજ્ય સરકાર પર છે. અમે ચોક્કસપણે તેના પર રાજ્ય સરકાર સાથે મળીને કામ કરીશું.

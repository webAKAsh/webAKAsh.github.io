ਜਿਵੇਂ ਕਿ ਤੁਸੀਂ ਜਾਣਦੇ ਹੋ ਅਨੰਤਪੁਰ ਪ੍ਰਸਤਾਵਿਤ ਬੈਂਗਲੁਰੂ-ਕੁਰਨੂਲ ਇੰਡਸਟਰੀਅਲ ਕੋਰੀਡੋਰ ਅਤੇ ਸੀਬੀਆਈਸੀ, ਯਾਨੀ ਚੇਨਈ-ਬੰਗਲੁਰੂ ਇੰਡਸਟਰੀਅਲ ਕੋਰੀਡੋਰ ਦਾ ਹਿੱਸਾ ਹੈ। ਕੀ ਅਨੰਤਪੁਰ ਨੂੰ NIMZ ਵਿੱਚ ਸ਼ਾਮਲ ਕਰਨ ਦਾ ਕੋਈ ਪ੍ਰਸਤਾਵ ਹੈ? ਕੀ ਇਹਨਾਂ 22 NIMZs ਨੂੰ ਪੂਰੀ ਤਰ੍ਹਾਂ ਲਾਗੂ ਕਰਨ ਲਈ ਕੋਈ ਕਾਰਜ ਯੋਜਨਾ ਜਾਂ ਸਮਾਂ ਸੀਮਾ ਨਿਰਧਾਰਤ ਕੀਤੀ ਗਈ ਹੈ? … ( ਰੁਕਾਵਟਾਂ ) ਸ਼੍ਰੀਮਤੀ ਨਿਰਮਲਾ ਸੀਤਾਰਮਨ: ਮੈਡਮ ਸਪੀਕਰ, ਮਾਨਯੋਗ। ਮੈਂਬਰ ਨੇ ਅਨੰਤਪੁਰ ਜ਼ਿਲ੍ਹੇ ਬਾਰੇ ਸਵਾਲ ਪੁੱਛੇ ਹਨ। … ( ਰੁਕਾਵਟਾਂ ) ਜਿਵੇਂ ਕਿ ਇਹ ਹੈ, ਅਸੀਂ ਜਾਣਦੇ ਹਾਂ ਕਿ ਦਿੱਲੀ-ਮੁੰਬਈ ਇੰਡਸਟਰੀਅਲ ਕੋਰੀਡੋਰ ਦੇ ਅਧੀਨ ਅੱਠ ਜ਼ੋਨ ਕਵਰ ਕੀਤੇ ਜਾ ਰਹੇ ਹਨ ਅਤੇ 14 ਹੋਰ, ਜੋ ਕਿ ਦਿੱਲੀ-ਮੁੰਬਈ ਇੰਡਸਟਰੀਅਲ ਕੋਰੀਡੋਰ ਤੋਂ ਬਾਹਰ ਹਨ, ਨੂੰ ਜ਼ੋਨਾਂ ਵਜੋਂ ਸਮਰਪਿਤ ਕੀਤਾ ਗਿਆ ਹੈ, ਪਰ ਕੋਈ ਨਵਾਂ ਪ੍ਰਸਤਾਵ ਨਹੀਂ ਹੈ। ਕਿਸੇ ਖਾਸ ਜ਼ਿਲ੍ਹੇ ਨੂੰ ਸ਼ਾਮਲ ਕਰੋ, ਜਿਵੇਂ ਕਿ ਮੈਂਬਰ ਨੇ ਅਨੰਤਪੁਰ ਦਾ ਸਮਰਥਨ ਕੀਤਾ ਹੈ। ਉਹ ਜਾਂ ਹੋਰਾਂ ਨੂੰ ਸੂਚੀ ਵਿੱਚ ਨਵੇਂ ਸ਼ਾਮਲ ਨਹੀਂ ਕੀਤਾ ਗਿਆ ਹੈ। … ( ਰੁਕਾਵਟਾਂ ) ਸ਼੍ਰੀ ਜੇ ਸੀ ਦਿਵਾਕਰ ਰੈਡੀ: ਮੈਡਮ, ਅਨੰਤਪੁਰ ਇੱਕ ਪਛੜਿਆ ਜ਼ਿਲ੍ਹਾ ਹੈ। ਇਹ ਬੈਂਗਲੁਰੂ ਦੇ ਨਾਲ ਹੈ ਅਤੇ ਬੈਂਗਲੁਰੂ ਦੇ ਨੇੜੇ ਹੈ। ਇਸ ਲਈ, ਮਾਨਯੋਗ. ਮੰਤਰੀ ਨੂੰ ਅਨੰਤਪੁਰ ਦੇ ਵਿਕਾਸ ਦੀ ਚਿੰਤਾ ਕਰਨੀ ਚਾਹੀਦੀ ਹੈ। … ( ਰੁਕਾਵਟਾਂ ) ਮੈਂ ਉਸ ਨੂੰ ਬੇਨਤੀ ਕਰਾਂਗਾ ਕਿ ਉਹ ਉੱਥੇ ਇੱਕ ਹੋਰ ਜ਼ੋਨ ਦਾ ਪ੍ਰਸਤਾਵ ਕਰੇ ਤਾਂ ਜੋ ਟੈਕਸਟਾਈਲ ਅਤੇ ਖੇਤੀਬਾੜੀ ਵਰਗੇ ਉਦਯੋਗ ਸਥਾਪਤ ਕਰਨ ਦਾ ਮੌਕਾ ਉਪਲਬਧ ਹੋਵੇ। ਇਸ ਲਈ, ਮੇਰੀ ਮਾਨਯੋਗ ਨੂੰ ਇਹ ਵਿਸ਼ੇਸ਼ ਬੇਨਤੀ ਹੈ. ਮੰਤਰੀ ਅਨੰਤਪੁਰ ਨੂੰ ਸ਼ਾਮਲ ਕਰਨ ਦੀ ਕੋਸ਼ਿਸ਼ ਕਰਨਗੇ। … ( ਰੁਕਾਵਟਾਂ ) ) ਮਾਨਯੋਗ ਸਪੀਕਰ: ਮੈਨੂੰ ਅਫ਼ਸੋਸ ਹੈ। … ( ਰੁਕਾਵਟਾਂ ) ਮਾਨਯੋਗ ਸਪੀਕਰ: ਸਦਨ ਸਵੇਰੇ 11.20 ਵਜੇ ਦੁਬਾਰਾ ਮੀਟਿੰਗ ਲਈ ਮੁਲਤਵੀ ਕਰ ਦਿੱਤਾ ਗਿਆ ਇਸ ਤੋਂ ਬਾਅਦ ਲੋਕ ਸਭਾ ਗਿਆਰਾਂ ਵੱਜ ਕੇ 20 ਮਿੰਟ ਤੱਕ ਮੁਲਤਵੀ ਕਰ ਦਿੱਤੀ ਗਈ ਘੜੀ ਦੇ. ਸਵਾਲਾਂ ਦੇ ਮੌਖਿਕ ਜਵਾਬ .... ਜਾਰੀ। ਸ਼੍ਰੀ ਜੇ.ਸੀ. ਦਿਵਾਕਰ ਰੈੱਡੀ: ਮੈਡਮ, ਮਾਣਯੋਗ ਮੰਤਰੀ ਅਨੰਤਪੁਰ ਨੂੰ ਚੰਗੀ ਤਰ੍ਹਾਂ ਜਾਣਦੇ ਹਨ। … ( ਰੁਕਾਵਟਾਂ ) HON. ਸਪੀਕਰ: ਨਹੀਂ, ਮੈਂ ਕਿਸੇ ਨੂੰ ਇਜਾਜ਼ਤ ਨਹੀਂ ਦੇ ਰਿਹਾ ਹਾਂ। … ( ਰੁਕਾਵਟਾਂ ) ਸ਼੍ਰੀ ਜੇ.ਸੀ. ਦਿਵਾਕਰ ਰੈਡੀ: ਅਨੰਤਪੁਰ ਇੱਕ ਪਛੜਿਆ ਇਲਾਕਾ ਹੈ। ਇਹ ਰਾਜਸਥਾਨ ਦੇ ਜੈਸਲਮੇਰ ਜ਼ਿਲ੍ਹੇ ਤੋਂ ਵੀ ਮਾੜੀ ਹੈ ਕਿਉਂਕਿ ਉਦਯੋਗਿਕ ਅਤੇ ਖੇਤੀਬਾੜੀ ਪੱਖੋਂ ਅਸੀਂ ਬਹੁਤ ਗਰੀਬ ਹਾਂ ਅਤੇ ਬਹੁਤ ਸਾਰੇ ਲੋਕ ਉੱਥੋਂ ਪਲਾਇਨ ਕਰ ਰਹੇ ਹਨ। HON. ਸਪੀਕਰ: ਤੁਹਾਡਾ ਸਵਾਲ ਕੀ ਹੈ?

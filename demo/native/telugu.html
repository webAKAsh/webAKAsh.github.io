ప్రశ్నలకు మౌఖిక సమాధానాలు (Q.501) … ( అంతరాయాలు ) శ్రీ జెసి దివాకర్ రెడ్డి: మేడమ్ స్పీకర్, నాకు ఈ అవకాశం ఇచ్చినందుకు చాలా ధన్యవాదాలు. … ( అంతరాయాలు ) మా గౌరవనీయుని డైనమిక్ నాయకత్వంలో ఆగస్ట్ హౌస్ మొత్తం నాతో అంగీకరిస్తుంది. ప్రధాన మంత్రి 'మేక్ ఇన్ ఇండియా' కల మరియు మా గౌరవ లక్ష్యం. ఆంధ్రప్రదేశ్ ముఖ్యమంత్రి - 'మేక్ ఇన్ ఆంధ్ర' కలను సాకారం చేయడానికి ఎల్లప్పుడూ కృషి చేసేవారు - నేషనల్ ఇండస్ట్రియల్ అండ్ మ్యానుఫ్యాక్చరింగ్ జోన్‌లు (NIMZs) భవిష్యత్తులో విజయవంతం అవుతాయని నేను ఆశిస్తున్నాను.… ( అంతరాయాలు ) నేషనల్ మ్యానుఫ్యాక్చరింగ్ పాలసీలో నేషనల్ ఇన్వెస్ట్‌మెంట్ మరియు మ్యానుఫ్యాక్చరింగ్ జోన్‌లలో తయారీ పరిశ్రమను మరియు దేశంలోని మొత్తం తయారీ పరిశ్రమను సులభతరం చేయడానికి డిస్పెన్సేషన్‌లను కలిగి ఉంది, తయారీ యూనిట్లు తమను తాము క్లస్టర్‌లుగా నిర్వహించుకునేలా చేయడంతో సహా. … ( అంతరాయాలు ) 2016-17 ఆర్థిక సంవత్సరానికి రూ. నేషనల్ మ్యానుఫ్యాక్చరింగ్ పాలసీ అమలు, NIMZల మాస్టర్ ప్లానింగ్ మరియు టెక్నాలజీ అక్విజిషన్ అండ్ డెవలప్‌మెంట్ ఫండ్ కింద 3.35 కోట్లు కేటాయించారు. … ( అంతరాయాలు ) గౌరవనీయులకు నా మొదటి అనుబంధ ప్రశ్న. మంత్రి గారు మీ ద్వారా మేడమ్ స్పీకర్ ఇది. ఈ NIMZలు ఎప్పుడు వెలుగు చూస్తాయి? ప్రతిపాదిత బెంగళూరు-కర్నూలు ఇండస్ట్రియల్ కారిడార్ మరియు CBIC, అంటే చెన్నై-బెంగళూరు పారిశ్రామిక కారిడార్‌లో అనంతపురం భాగమని మీకు తెలుసు. అనంతపురం నిమ్‌జెడ్‌లలో చేర్చే ప్రతిపాదన ఏమైనా ఉందా? ఈ 22 NIMZల పూర్తి అమలు కోసం ఏదైనా కార్యాచరణ ప్రణాళిక లేదా కాలపరిమితి నిర్ణయించబడిందా? … ( అంతరాయాలు ) శ్రీమతి నిర్మలా సీతారామన్: మేడమ్ స్పీకర్, గౌరవనీయులు. అనంతపురం జిల్లాపై సభ్యులు ప్రశ్నలు అడిగారు. … ( అంతరాయాలు ) ఢిల్లీ-ముంబై ఇండస్ట్రియల్ కారిడార్ కింద ఎనిమిది జోన్‌లు కవర్ చేయబడుతున్నాయని మరియు ఢిల్లీ-ముంబై ఇండస్ట్రియల్ కారిడార్ వెలుపల ఉన్న 14 జోన్‌లను జోన్‌లుగా అంకితం చేశారని మాకు తెలుసు, అయితే కొత్త ప్రతిపాదన లేదు సభ్యుడు అనంతపురంకు మద్దతు ఇవ్వడం వంటి నిర్దిష్ట జిల్లాను చేర్చండి. అది లేదా ఇతరులు జాబితాలో కొత్తగా చేర్చబడలేదు. … ( అంతరాయాలు ) శ్రీ జేసీ దివాకర్ రెడ్డి: మేడం, అనంతపురం వెనుకబడిన జిల్లా. ఇది బెంగళూరు పక్కన మరియు బెంగళూరుకు సమీపంలో ఉంది. కాబట్టి, గౌరవనీయులు. అనంతపురం అభివృద్ధిపై మంత్రికి శ్రద్ధ ఉండాలి. … ( అంతరాయాలు ) అక్కడ మరో జోన్‌ను ప్రతిపాదించమని నేను ఆమెను అభ్యర్థిస్తున్నాను, తద్వారా టెక్స్‌టైల్ మరియు వ్యవసాయం వంటి పరిశ్రమలను స్థాపించడానికి అవకాశం ఉంది. కాబట్టి, ఇది గౌరవనీయులకు నా ప్రత్యేక విన్నపం. అనంతపురంలో చేర్చేందుకు మంత్రి ప్రయత్నించారు. … ( అంతరాయాలు ) ) HON. స్పీకర్: నన్ను క్షమించండి. … ( అంతరాయాలు ) HON. స్పీకర్: మళ్లీ ఉదయం 11.20 గంటలకు సభ వాయిదా పడింది ఆ తర్వాత లోక్‌సభ పదకొండు దాటి ఇరవై నిమిషాల వరకు వాయిదా పడింది గడియారం యొక్క. ప్రశ్నలకు మౌఖిక సమాధానాలు .... కొనసాగింపు. శ్రీ జేసీ దివాకర్ రెడ్డి: మేడమ్, గౌరవనీయులైన మంత్రికి అనంతపురం గురించి బాగా తెలుసు. … ( అంతరాయాలు ) HON. స్పీకర్: లేదు, నేను ఎవరినీ అనుమతించడం లేదు. … ( అంతరాయాలు ) శ్రీ జేసీ దివాకర్ రెడ్డి: అనంతపురం వెనుకబడిన ప్రాంతం. ఇది రాజస్థాన్‌లోని జైసల్మేర్ జిల్లా కంటే అధ్వాన్నంగా ఉంది, ఎందుకంటే పారిశ్రామికంగా మరియు వ్యవసాయపరంగా మేము చాలా పేదవాళ్లం మరియు చాలా మంది ప్రజలు అక్కడి నుండి వలసపోతున్నారు. HON. స్పీకర్: మీ ప్రశ్న ఏమిటి? శ్రీ జేసీ దివాకర్ రెడ్డి: అవును మేడమ్, నేను అడుగుతున్నాను. బెంగళూరు, చెన్నై తదితర ప్రాంతాలకు వలస వెళ్తున్నారు. రోజురోజుకు అక్కడి నుంచి వలస వెళ్లే వారి సమస్య పెరుగుతోంది, అయితే అదే సమయంలో మనకు చాలా వృధాగా ఉన్న భూమి మరియు అక్కడ మంచి భూమి ఉంది. నా విన్నపం ఇది మాత్రమే, మీ ద్వారా, మా గౌరవనీయులకు. మంత్రి. ఆమె ప్రకాశం, చిత్తూరు జిల్లాలను గుర్తించి, అక్కడ రెండు కేంద్రాలను అందించారు. ఈ జిల్లాలన్నింటి కంటే నా జిల్లా అధ్వాన్నంగా ఉందని నా విన్నపం. మీరు మా కోసం మరో కేంద్రం ఎందుకు ఇవ్వరు? ప్రభుత్వం ఏమీ కోల్పోదు, అయితే అది మనం ఆర్థికంగా మరియు ఆర్థికంగా అభివృద్ధి చెందడానికి అవకాశం ఇస్తుంది. దయచేసి దాని కోసం నా జిల్లాను పరిగణించండి. శ్రీమతి నిర్మలా సీతారామన్: మేడమ్ స్పీకర్, గౌరవనీయమైన అంశం. అనంతపురం జిల్లా వేగంగా ఎడారిగా మారుతున్నందున సభ్యులు లేవనెత్తడం ఖచ్చితంగా చాలా ముఖ్యమైనది. ఈ జిల్లాకు పారిశ్రామికంగా మరియు ఇతరత్రా అవకాశాలను కల్పించాల్సిన అవసరం ఉన్నందున ఈ జిల్లాకు ఇబ్బంది ఉందని మేము గుర్తించాము. కానీ రాష్ట్ర ప్రభుత్వం మాకు ఏదైనా ప్రతిపాదన పంపితే, అది వెళ్లవలసిన శాఖను బట్టి మనం దానిని పరిశీలించవచ్చు. కాబట్టి, దానిని రాష్ట్ర ప్రభుత్వం చేపట్టాలి. దీనిపై రాష్ట్ర ప్రభుత్వంతో కచ్చితంగా కలిసి పనిచేస్తాం.

கேள்விகளுக்கான வாய்வழி பதில்கள் (கே.501) … ( குறுக்கீடுகள் ) ஸ்ரீ ஜே.சி. திவாகர் ரெட்டி: சபாநாயகர் மேடம், எனக்கு இந்த வாய்ப்பை வழங்கியதற்கு மிக்க நன்றி. … ( குறுக்கீடுகள் ) எங்கள் மாண்புமிகு தலைவரின் ஆற்றல்மிக்க தலைமையின் கீழ் முழு ஆகஸ்ட் சபையும் என்னுடன் உடன்படும். 'மேக் இன் இந்தியா' கனவு மற்றும் நமது மாண்புமிகு பிரதமர். ஆந்திரப் பிரதேசத்தின் முதலமைச்சர் - 'மேக் இன் ஆந்திரா' என்ற கனவை எப்போதும் நனவாக்க பாடுபடுபவர் - தேசிய தொழில்துறை மற்றும் உற்பத்தி மண்டலங்கள் (NIMZs) எதிர்காலத்தில் வெற்றிகரமாக இருக்கும் என்று நம்புகிறேன்.… ( குறுக்கீடுகள் ) தேசிய உற்பத்திக் கொள்கையானது, தேசிய முதலீடு மற்றும் உற்பத்தி மண்டலங்களில் உற்பத்தித் தொழிலை எளிதாக்குவதற்கும், உற்பத்தி அலகுகள் தங்களைக் குழுக்களாக ஒழுங்கமைத்துக் கொள்ள உதவுவது உட்பட, ஒட்டுமொத்த நாட்டிலுள்ள உற்பத்தித் தொழிலுக்கும் உதவும். … ( குறுக்கீடுகள் ) 2016-17ஆம் நிதியாண்டுக்கு ரூ. 3.35 கோடி தேசிய உற்பத்திக் கொள்கையை செயல்படுத்துதல், NIMZ களின் மாஸ்டர் பிளான்னிங் மற்றும் தொழில்நுட்ப கையகப்படுத்துதல் மற்றும் மேம்பாட்டு நிதி ஆகியவற்றின் கீழ் ஒதுக்கப்பட்டுள்ளது. … ( குறுக்கீடுகள் ) மாண்புமிகு அவர்களுக்கு எனது முதல் துணைக் கேள்வி. அமைச்சரே, உங்கள் மூலமாக சபாநாயகர் மேடம் இது. இந்த NIMZகள் எப்போது வெளிச்சத்தைக் காணும்? அனந்தபூர் முன்மொழியப்பட்ட பெங்களூரு-கர்னூல் தொழில்துறை தாழ்வாரம் மற்றும் சிபிஐசி, அதாவது சென்னை-பெங்களூரு தொழில்துறை தாழ்வாரத்தின் ஒரு பகுதி என்பது உங்களுக்குத் தெரியும். NIMZ களில் அனந்தபூரை சேர்க்க ஏதேனும் முன்மொழிவு உள்ளதா? இந்த 22 NIMZகளை முழுமையாக செயல்படுத்துவதற்கு ஏதேனும் செயல் திட்டம் அல்லது காலக்கெடு நிர்ணயிக்கப்பட்டுள்ளதா? … ( குறுக்கீடுகள் ) ஸ்ரீமதி நிர்மலா சீதாராமன்: மேடம் சபாநாயகர், மாண்புமிகு. அனந்தபூர் மாவட்டம் குறித்து உறுப்பினர் கேள்விகள் கேட்டுள்ளார். … ( குறுக்கீடுகள் ) டெல்லி-மும்பை தொழில்துறை தாழ்வாரத்தின் கீழ் எட்டு மண்டலங்கள் மூடப்பட்டு வருகின்றன என்பதையும், டெல்லி-மும்பை தொழில்துறை தாழ்வாரத்திற்கு வெளியே உள்ள 14 மண்டலங்கள் மண்டலங்களாக அர்ப்பணிக்கப்பட்டுள்ளன என்பதையும் நாங்கள் அறிவோம், ஆனால் அதற்கான புதிய திட்டம் எதுவும் இல்லை. உறுப்பினர் அனந்தபூரை ஆதரித்தது போன்ற ஒரு குறிப்பிட்ட மாவட்டத்தை உள்ளடக்கியது. அது அல்லது மற்றவர்கள் பட்டியலில் புதிதாக சேர்க்கப்படவில்லை. … ( குறுக்கீடுகள் ) ஸ்ரீ ஜே.சி. திவாகர் ரெட்டி: மேடம், அனந்தபூர் ஒரு பின்தங்கிய மாவட்டம். இது பெங்களூரு பக்கத்திலும், பெங்களூருக்கு அருகில் உள்ளது. எனவே, கௌரவ. அனந்தபூரின் வளர்ச்சியில் அமைச்சர் அக்கறை காட்ட வேண்டும். … ( குறுக்கீடுகள் ) ஜவுளி மற்றும் விவசாயம் போன்ற தொழில்களை அமைப்பதற்கான வாய்ப்பு கிடைக்கும் வகையில் அங்கு மேலும் ஒரு மண்டலத்தை முன்மொழியுமாறு நான் அவரிடம் கேட்டுக்கொள்கிறேன். எனவே, மாண்புமிகு அவர்களுக்கு இது எனது சிறப்பு வேண்டுகோள். அனந்தபூரை சேர்க்க முயற்சி செய்ய அமைச்சர். … ( குறுக்கீடுகள் ) ) கௌரவ. பேச்சாளர்: மன்னிக்கவும். … ( குறுக்கீடுகள் ) கௌரவ. சபாநாயகர்: காலை 11.20 மணிக்கு மீண்டும் கூடுவதற்காக சபை ஒத்திவைக்கப்பட்டது இதனையடுத்து மக்களவை பதினொன்றை கடந்த இருபது நிமிடங்களுக்கு ஒத்திவைக்கப்பட்டது கடிகாரத்தின். கேள்விகளுக்கான வாய்வழி பதில்கள் .... தொடர்ச்சி. ஸ்ரீ ஜே.சி. திவாகர் ரெட்டி: மேடம், மரியாதைக்குரிய அமைச்சருக்கு அனந்தபூரை நன்றாகத் தெரியும். … ( குறுக்கீடுகள் ) HON. சபாநாயகர்: இல்லை, நான் யாரையும் அனுமதிக்கவில்லை. … ( குறுக்கீடுகள் ) ஸ்ரீ ஜே.சி. திவாகர் ரெட்டி: அனந்தபூர் ஒரு பின்தங்கிய பகுதி. ராஜஸ்தானில் உள்ள ஜெய்சால்மர் மாவட்டத்தை விட இது மோசமானது, ஏனெனில் தொழில்துறை மற்றும் விவசாயம் வாரியாக நாங்கள் மிகவும் ஏழ்மையானவர்கள், மேலும் பலர் அங்கிருந்து இடம்பெயர்கின்றனர். HON. பேச்சாளர்: உங்கள் கேள்வி என்ன? ஸ்ரீ ஜே.சி. திவாகர் ரெட்டி: ஆமாம், மேடம், நான் அதைக் கேட்கிறேன். பெங்களூரு, சென்னை உள்ளிட்ட இடங்களுக்கு இடம் பெயர்ந்து வருகின்றனர். நாளுக்கு நாள், மக்கள் அங்கிருந்து இடம்பெயர்வதற்கான பிரச்சினை அதிகரித்து வருகிறது, ஆனால் அதே நேரத்தில் எங்களுக்கு ஏராளமான தரிசு நிலங்களும் நல்ல நிலங்களும் உள்ளன. எனது வேண்டுகோள் இது மட்டுமே, உங்கள் மூலம், எங்கள் மாண்புமிகு. அமைச்சர். அவர் பிரகாசம் மற்றும் சித்தூர் மாவட்டங்களை அங்கீகரித்துள்ளார் மற்றும் அங்கு அவர் வழங்கிய இரண்டு மையங்கள். இந்த அனைத்து மாவட்டங்களையும் விட எனது மாவட்டம் மோசமாக உள்ளது என்பதே எனது வேண்டுகோள். எங்களுக்காக இன்னும் ஒரு மையத்தை ஏன் கொடுக்கக் கூடாது? அரசாங்கம் எதனையும் இழக்காது அதேசமயம் பொருளாதார ரீதியாகவும் நிதி ரீதியாகவும் நாம் முன்னேற்றமடைய வாய்ப்பளிக்கும். அதற்கு எனது மாவட்டத்தை கருத்தில் கொள்ளவும். ஸ்ரீமதி நிர்மலா சீதாராமன்: சபாநாயகர் மேடம், இந்த விவகாரம் மாண்புமிகு. அனந்தபூர் மாவட்டம் வேகமாக பாலைவனமாக மாறி வருவதால், உறுப்பினர் எழுப்பியிருப்பது நிச்சயமாக மிகவும் முக்கியமானது. இம்மாவட்டம் தொழில்ரீதியாகவும் மற்றபடி அதன் மக்களுக்கு வாய்ப்புகளை வழங்க வேண்டியிருப்பதால் இம்மாவட்டத்திற்கு சிரமம் இருப்பதை நாங்கள் அறிவோம். ஆனால், மாநில அரசு எங்களுக்கு ஏதேனும் ஒரு திட்டத்தை அனுப்பினால், அது எந்தத் துறைக்குச் செல்ல வேண்டும் என்பதைப் பொறுத்து அதை நாங்கள் பரிசீலிக்கலாம். எனவே, அதை மாநில அரசு தான் எடுக்க வேண்டும். கண்டிப்பாக மாநில அரசுடன் இணைந்து செயல்படுவோம்.
